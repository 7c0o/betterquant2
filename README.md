## 编译
&emsp;&emsp;目前在Ubuntu 20.04上编译通过，系统本身是跨平台的，只是在Ubuntu 20.04 LTS完善了初始化脚本，我是在 root 用户下测试的，请切换到 root 用户。
```bash
   cd betterquant2 && bash init-sys.sh && bash build-sln-all.sh
```
